# Legacy Source

Este directorio conserva la aplicacion `intelicoop` original como referencia historica.

No debe tratarse como codigo activo del modulo SIPET.

Uso permitido:

- consultar comportamiento anterior
- reutilizar logica todavia no migrada
- comparar contratos o modelos del sistema original

Uso no permitido:

- agregar nuevas funcionalidades aqui
- mantener este arbol como segunda app viva dentro de SIPET
