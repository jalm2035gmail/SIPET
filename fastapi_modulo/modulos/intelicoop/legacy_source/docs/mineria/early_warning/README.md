# Early Warning System 1.7

Objetivo:
- Anticipar deterioro con alertas tempranas accionables.

Comando:
- `python manage.py alertas_tempranas_early_warning_1_7`

Salidas:
- `docs/mineria/early_warning/01_alertas_tempranas.csv`
- `docs/mineria/early_warning/02_semaforos_dashboard.csv`
- `docs/mineria/early_warning/03_notificaciones_internas.jsonl`
- `docs/mineria/early_warning/01_early_warning_system.md`

Cobertura:
- Caida fuerte de ahorro
- Cambio brusco de patron de pago
- Primer atraso (mora temprana)
- Concentracion de mora (producto/sucursal/ejecutivo)
- Deterioro de cobertura
- Notificaciones internas (correo / whatsapp interno)
- Semaforos para dashboard
